# 📇 Administración de Agenda de Contactos

Este es un proyecto de consola desarrollado en **Python** que permite gestionar una agenda de contactos de forma eficiente. El sistema incluye validaciones de datos para asegurar que la información guardada sea correcta.

## 🚀 Características
- **Agregar contactos**: Con validación de nombre, teléfono (solo números) y formato de correo electrónico.
- **Listar contactos**: Visualización organizada de todos los registros.
- **Buscar contactos**: Búsqueda por nombre o teléfono.
- **Editar contactos**: Permite actualizar la información de un contacto existente.
- **Eliminar contactos**: Borrado selectivo mediante índices.

## 🛠️ Tecnologías utilizadas
- **Lenguaje:** Python 3.x
- **Entorno:** Google Colab / CLI (Interfaz de línea de comandos)

## 📋 Requisitos
No requiere librerías externas. Solo necesitas tener instalado Python o ejecutarlo en un entorno como Google Colab o VS Code.
